# Feast-Hub User Guide

## Purpose
Feast-Hub helps users manage recipes and pantry ingredients in one place.

## Main Tasks

### View Recipes
Select option 1 to view all available recipes.

### Search Recipe
Select option 2 and enter a keyword such as `pasta` or `salad`.

### Add Recipe
Select option 3. Enter the title, ingredients separated by `|`, and cooking instructions.

### Rate Recipe
Select option 4. Enter the recipe ID and a rating from 1 to 5.

### Manage Pantry
Use options 5 and 6 to view or add ingredients.

### Backup Data
Select option 10 to create a backup of recipes, pantry data, and support logs.

## Basic Troubleshooting

- If the app does not start, confirm Java is installed.
- If records do not show, confirm the `data` folder exists.
- If backup fails, confirm the project folder has write permission.
